# X/Twitter 自动发推方案研究报告

## 一、X API 官方方案

### 1.1 定价结构（2025年最新）

| 套餐 | 价格 | 发推数量 | 说明 |
|------|------|----------|------|
| **Free（免费）** | $0 | ❌ 仅读取 | 只能读取推文，**无法发推** |
| **Basic（基础）** | $200/月 | 50,000条/月 | 发推+读取的最低门槛 |
| **Pro（专业）** | $5,000/月 | 10,000,000条/月 | 高流量应用 |
| **Pay-per-use（按量付费）** | 2025年10月推出试点 | 灵活计费 | 新选项，可能更便宜 |

### 1.2 官方API使用流程

1. 访问 [developer.x.com](https://developer.x.com) 注册开发者账号
2. 创建 App 获取 API Key
3. 选择套餐（免费套餐无法发推！）
4. 使用 OAuth 2.0 认证
5. 调用 POST /2/tweets 接口发推

### 1.3 结论

> ❌ **免费方案不可行**：X 已取消免费发推 API，必须付费

## 二、第三方工具方案

### 2.1 IFTTT（推荐免费方案）

- **官网**: ifttt.com
- **免费版**: 基础功能，检查频率每小时1次
- **Pro版**: $2.5/月起，检查频率每5分钟
- **功能**: 可设置定时发推、跨平台同步

**优势**: 免费可用，适合萌宠图片定时发布

### 2.2 Highperformr

- **定位**: 面向创作者和企业的 AI 驱动工具
- **功能**: 内容生成、发布时间优化、线程支持
- **价格**: 付费工具，具体价格需官网查询

### 2.3 Apify

- **价格**: 部分服务 $10/月
- **功能**: Twitter 发推/回复 API

### 2.4 其他工具

## 三、免费/低成本方案推荐

### 🥇 最佳免费方案：IFTTT

**设置步骤：**
1. 注册 ifttt.com 账号
2. 连接 Twitter 账号
3. 创建 Applet：
   - 触发条件：Date & Time（定时）或其他来源
   - 执行操作：Post a tweet
4. 设置每日发推时间

**限制：** 免费版每小时检查一次

---

### 🥈 低成本方案：Python 脚本 + 付费 API

如果愿意投入 $100/月左右：

```python
# 伪代码示例
import tweepy

client = tweepy.Client(
    bearer_token=BEARER_TOKEN,
    consumer_key=API_KEY,
    consumer_secret=API_SECRET,
    access_token=ACCESS_TOKEN,
    access_token_secret=ACCESS_TOKEN_SECRET
)

# 发推
response = client.create_tweet(text="萌宠图片分享 🐱")
```

## 四、针对"萌宠自动发推"的建议方案

### 方案A：IFTTT（完全免费）
- **成本**: $0
- **操作**: 
  1. 准备萌宠图片素材库
  2. 用 IFTTT 设置每日定时任务
  3. 从 Google Drive/Photos 获取图片发推

### 方案B：自建脚本 + Basic API（推荐）
- **成本**: ~$200/月（约 ¥1400/月）
- **优点**: 完全可控，稳定可靠
- **适合**: 长期运营

### 方案C：第三方调度服务 + Basic API
- **成本**: $200/月 + 少量工具费
- **组合**: n8n / Zapier 调度 + X API
